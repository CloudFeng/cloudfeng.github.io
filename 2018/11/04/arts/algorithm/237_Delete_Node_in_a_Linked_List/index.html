237. Delete Node in a Linked List

2018-11-04 Algorithm

Algorithm

本周做的算法题是 237. Delete Node in a Linked List。

问题描述

写一个函数删除单链表中得一个节点（非尾节点），并假定给定的节点是可以获取的。

例子：给定一个单链表：4 -> 5 -> 1 -> 9

  1. 若删除节点 5， 链表变为： 4 -> 1 -> 9
  2. 若删除节点 1， 链表变为： 4 -> 5 -> 9

思路

  1. 将要删除的节点之后向节点的值赋值到删除的节点移动，后续的节点补上，若删除是倒数第二个节点，时间复杂度为 O(n-1), 其中n为链表的长度。
  1. 将要删除的节点的值替换为后面一个节点的值，然后删除节点下一个指针，指向删除节点的下一个节点再下一个节点。时间复杂度为 O(n)。
