linux 磁盘管理的小命令

工作上经常要清理一些文件以及查看磁盘的使用量，本周分享一下常用的几个磁盘管理命令。

  • df：列出文件系统的整体磁盘使用量
  • du：检查磁盘空间使用量，文件和目录磁盘使用空间
  • fdisk：用于磁盘分区
